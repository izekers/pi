## 项目概述

Pi 是一个本地运行的 Coding Agent，通过自然语言帮助用户在本地仓库中完成编码任务。支持多种 LLM 提供商（Anthropic、Azure OpenAI 等），终端 UI 基于自研的 TUI 渲染引擎。

**技术栈：** TypeScript + Node.js (≥22.19.0) + 自研 TUI
