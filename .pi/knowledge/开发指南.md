# 开发指南

## 环境要求

- Node.js ≥ 22.19.0
- npm（随 Node.js 一起安装）

## 本地开发流程

```bash
# 1. 安装依赖（跳过编译脚本）
npm install --ignore-scripts

# 2. 构建所有包
npm run build

# 3. 链接 pi 命令到全局
cd packages/coding-agent && npm link

# 4. 验证安装
pi --version
```

## 包构建顺序

```
tui → ai → agent → coding-agent
```

依赖链是单向的，前一个包构建完才能构建下一个。

## 运行测试

```bash
# 运行所有包的测试
npm run test --workspaces

# 或在单个包内运行
cd packages/coding-agent && npm run test
```

## 代码检查

```bash
npm run check
```

包括 Biome格式化检查、TypeScript 类型检查、依赖版本检查等。
