# 配置说明

## .pi 目录

`.pi/` 是 Pi 的配置目录，位于用户运行 `pi` 的工作目录或主目录下。

```
.pi/
├── agent/          # Agent 配置
├── extensions/     # 扩展程序
├── git/            # Git 钩子配置
├── npm/            # npm知识库（Obsidian Vault）
├── knowledge/      # 项目知识库（Obsidian Vault）
├── prompts/        # 提示词配置
└── skills/         # 技能配置
```

## Pi配置文件

- `AGENTS.md` — 定义 Agent 行为和可用工具
- `.pi/config/` — 运行时配置
