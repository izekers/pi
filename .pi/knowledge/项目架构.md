# 项目架构

## Monorepo 结构

```
pi-monorepo/ ← 根目录（主项目）
├── package.json                 ← workspaces 声明，管理所有子包
├── packages/
│   ├── agent/                   ← @earendil-works/pi-agent-core
│   ├── ai/                      ← @earendil-works/pi-ai
│   ├── tui/                     ← @earendil-works/pi-tui
│   └── coding-agent/            ← @earendil-works/pi-coding-agent（CLI入口）
├── node_modules/                ← 共享依赖（npm workspaces）
└── scripts/                     ← 构建/发布脚本
```

##四个核心包

| 包 | 版本 | 说明 |
|----|------|------|
| `@earendil-works/pi-agent-core` | 0.78.1 | Agent 核心：状态管理、工具抽象、传输层 |
| `@earendil-works/pi-ai` | 0.78.1 | LLM 统一接口：Anthropic、Azure OpenAI 等 |
| `@earendil-works/pi-tui` | 0.78.1 | 终端 UI 引擎：差分渲染 |
| `@earendil-works/pi-coding-agent` | 0.78.1 | CLI 应用：包含 `pi` 命令入口 |

## 依赖关系

```
pi-coding-agent
├── pi-tui        (UI 渲染)
├── pi-ai         (LLM 调用)
└── pi-agent-core (核心逻辑)
```

## 安装与启动

```bash
# 安装依赖
npm install --ignore-scripts

# 构建所有包
npm run build

# 全局链接 pi 命令
cd packages/coding-agent && npm link

# 验证
pi --version
```
